# 九州六日自駕旅行網站

一個以 HTML、CSS 與 JavaScript 製作的九州六日自駕旅行網站。

## 專案介紹

本網站用於規劃與整理日本九州六日自駕旅遊行程，將每日景點、住宿、租車資訊、旅遊建議與行前準備事項整合於單一網站中，方便旅途中使用手機或平板即時查閱。

## 功能特色

- 📅 六天完整行程規劃
- 🗺️ 景點資訊與 Google Maps 導航連結
- 🚗 租車資訊整理
- 🏨 住宿資訊整理
- 🎒 行前檢查清單
- 📱 響應式設計，支援手機瀏覽
- 🍜 餐廳與美食推薦
- 🌋 九州熱門景點資訊整合
- 💰 記帳幫手展示頁
  - 若需要啟用旅伴共用記帳、支出紀錄、費用分攤與雲端同步功能，歡迎私訊聯繫。

## 技術架構

- HTML5
- CSS3
- JavaScript

## 使用方式

1. 開啟網站首頁
2. 透過左側選單切換功能頁面
3. 查看每日行程安排
4. 使用 Google Maps 連結進行導航
5. 依照行前清單確認旅遊準備事項

## 行程路線

福岡 → 熊本 → 高千穗 → 阿蘇 → 由布院 → 別府 → 太宰府 → 福岡

## 專案目的

透過網頁化方式整合旅遊資訊，取代紙本行程表與多個筆記 App，讓旅途中能快速查詢景點、住宿、導航及相關旅遊資訊。

## 備註

本網站目前為公開展示版本，部分進階功能僅保留介面展示。

如需啟用以下功能，歡迎私訊聯繫：

- 旅伴共用記帳
- 支出紀錄管理
- 自動分攤計算
- Firebase 雲端同步
- Google 帳號登入
- 多人協作旅遊管理

# Kyushu Road Trip Planner

A responsive travel planning website for a 6-day self-driving trip in Kyushu, Japan.

## Features

- Daily itinerary management
- Google Maps integration
- Accommodation information
- Rental car information
- Travel checklist
- Mobile-friendly responsive design
- Restaurant and local food recommendations
- Popular Kyushu attractions
- Expense tracker preview page
  - Shared expense tracking, bill splitting, and cloud synchronization features are available upon request.

## Technology Stack

- HTML5
- CSS3
- JavaScript

## Route

Fukuoka → Kumamoto → Takachiho → Aso → Yufuin → Beppu → Dazaifu → Fukuoka

## Notes

This repository contains the public version of the travel planner.

Advanced features such as shared expense tracking, bill splitting, Firebase cloud database, Google account authentication, and multi-user collaboration have been removed from the public release.

If you are interested in enabling these features for your own travel project, feel free to contact me for customization and implementation.
