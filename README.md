# 九州六日自駕旅行網站

🔗 線上預覽（Live Demo）

https://firelee1204.github.io/trip/

一個以 HTML、CSS 與 JavaScript 製作的九州六日自駕旅行網站。

## 專案介紹

本網站用於規劃與整理日本九州六日自駕旅遊行程，將每日景點、住宿、租車資訊、旅遊建議與行前準備事項整合於單一網站中，方便旅途中使用手機或平板即時查閱。

## 功能特色

- 📅 六天完整行程規劃
- 🗺️ 景點資訊與 Google Maps 導航連結
- 🚗 租車資訊整理
- 🏨 住宿資訊整理
- 🎒 行前檢查清單
- 📱 響應式設計，支援手機瀏覽
- 🍜 餐廳與美食推薦
- 🌋 九州熱門景點資訊整合
- 💰 記帳幫手展示頁

### 客製化功能

如需啟用以下功能，歡迎私訊聯繫：

- 多人共用記帳
- 旅伴費用分攤
- Firebase 雲端同步
- Google 帳號登入
- 多人協作旅遊管理
- 客製化旅遊規劃網站

## 技術架構

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 使用方式

1. 開啟網站首頁
2. 透過左側選單切換功能頁面
3. 查看每日行程安排
4. 使用 Google Maps 連結進行導航
5. 依照行前清單確認旅遊準備事項

## 行程路線

福岡 → 熊本 → 高千穗 → 阿蘇 → 由布院 → 別府 → 太宰府 → 福岡

## 專案目的

透過網頁化方式整合旅遊資訊，取代紙本行程表與多個筆記 App，讓旅途中能快速查詢景點、住宿、導航及相關旅遊資訊。
